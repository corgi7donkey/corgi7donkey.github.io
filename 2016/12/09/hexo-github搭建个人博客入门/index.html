博客搭建系列

hexo-github搭建个人博客入门

个人博客搭建入门系列一

  • 系统：Windows8.1
  • 需要安装的软件有：Git，Node.js，Hexo，Sublime_Text(可选用于查看内容)
  • 注册github账号

搭建流程

一、安装搭建本地Hexo博客

1，下载安装软件

去官网下载Git，Node.js和Sublime_Text并安装，一直next就可以了。

2，验证是否正确安装

进入cmd，输入下面命令查看版本。可以查看安装成功。

git –version
node -v

3，安装Hexo

(1) 在电脑上新建一个blog文件夹，右击进入git bash

(2) 安装hexo为了避免出现类似情况，使用淘宝NPM镜像,输入以下命令等待安装完成

npm install -g cnpm –registry=https://registry.npm.taobao.org

(3) 使用淘宝NPM安装Hexo

cnpm install -g hexo-cli

(4) 继续下面命令

cnpm install hexo –save

(5) 安装成功后，验证是否正确

hexo -v

4，本地运行hexo

（1）初始化hexo

hexo init

（2）安装生成器

cnpm install

（3）以后要在本地运行博客只要输入该命令即可

hexo s -g

（4）打开浏览器，输入localhost:4000,就可以在本地看到你的个人博客了

（5）停止运行

按住Ctrl+C键即可停止

二、搭建github博客

1，注册github

github注册比较简单，就不在这里啰嗦了。。。

2，新建仓库

（1）新建仓库方法1，点击start a project 方法2， 点击New repository

（2）输入Repository name:yourname.github.io(yourname与你的注册用户名一致,这个就是你博客的域名了)

（3）到此github仓库创建完成

3，启用并设置

（1）点击右边的“Setting”菜单进入设置,点击”Launch automatic page generator”

（2）点击底部的”Continue to layouts”

（3）最后点击”Publish page”,发布github默认生成的一个静态站点

（4）验证邮箱 点击个人设置

（5）点击”Send verification Email”发送验证邮件

（6）进入你的邮箱，查收验证邮件进行验证

三、将Hexo与github博客联系在一起

1,配置信息

使用安装好的Sublime_Text编辑器，打开blog/_config.yml文件，进行配置。下面是我的配置,关键的地方用红色字体 标记了

#博客名称
title: CorgiDonkey博客
#副标题
subtitle: 蜗牛的坚持
#简介
description: 学习记录
#博客作者
author: CorgiDonkey
#博客语言
language: zh-Hans
#时区
timezone:

#博客地址,与申请的GitHub一致
url: http://corgi7donkey.github.io
root: /
#博客链接格式
permalink: :year/:month/:day/:title/
permalink_defaults:

source_dir: source
public_dir: public
tag_dir: tags
archive_dir: archives
category_dir: categories
code_dir: downloads/code
i18n_dir: :lang
skip_render:

new_post_name: :title.md # File name of new posts
default_layout: post
titlecase: false # Transform title into titlecase
external_link: true # Open external links in new tab
filename_case: 0
render_drafts: false
post_asset_folder: false
relative_link: false
future: true
highlight:
enable: true
line_number: true
auto_detect: true
tab_replace:

default_category: uncategorized
category_map:
tag_map:

#日期格式
date_format: YYYY-MM-DD
time_format: HH:mm:ss

#分页，每页文章数量
per_page: 10
pagination_dir: page

#博客主题
theme: next

#发布设置
deploy:
type: git
#elfwalk改为你的github用户名
repository: https://github.com/corgi7donkey/corgi7donkey.github.io.git
branch: master

2,发布博客

（1）设置git身份信息

$ git config –global user.name “你的用户名”
$ git config –global user.email “你的邮箱”

（2）安装hexo git插件

$ cnpm install hexo-deployer-git –save

（3）发布更新博客 发布时需要输入github的帐号和密码，输入密码时是看不到自己输入的内容的

$ hexo d -g
