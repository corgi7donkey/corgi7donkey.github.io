Activity 启动模式

Activity启动模式分为四种：Standard，SingleTop，SingleTask和SingleInstance。可以在AndroidManifest.xml中通过给标签指定android：launchMode属性。

（1）standard

默认的启动模式，启动新的Activity后，系统就会将其放入返回栈的栈顶。在启动Activity时，不管返回栈中是否有此Activity，都会新建一个放到栈顶。

（2）singleTop

顾名思义：唯一栈顶。也就是说当启动Activity时如果发现栈顶是该Activity，则可以直接使用，不创建活动的实例。

（3）singleTask

顾名思义：唯一任务（Activity）。当启动Activity时，系统会先检查返回栈中是否有该Activity，如果有就使用该实例，并把它之上的所有Activity出栈，如果没有就新建一个实例。

（4）singleInstance

启动一个新的返回栈存放该Activity。
