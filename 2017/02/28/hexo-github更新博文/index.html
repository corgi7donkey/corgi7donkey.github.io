博文更新的常用命令

一、只更新新博文

（1）新建博文

hexo new "博文名"

（2）文章标题，日期，标签，分类

title: MVVM模式快速入门
date: 2015-11-13 15:40:25
tags: activity
categories: android

（3）文章写好直接执行下面命令即可直接发布文章

hexo d -g

（4）如果发布时候出现错误

ERROR Deployer not found: git

（5）执行

npm install hexo-deployer-git --save

二、更新所有博文

（1）清空博文，这步很关键

hexo clean

（2）生成博文(若要本地预览就先执行 hexo server)

hexo generate

（3）发布博文

hexo deploy
