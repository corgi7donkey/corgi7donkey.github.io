AndroidManifest.xml配置

配置接收优先级android:priority=”10”
数越大优先级越高（取值范围：-1000~10000）
优先级接收顺序：

  • 优先级高的先接收
  • 同优先级的动静态广播接收器，动态优先于静态
  • 同优先级的动态广播接收器 或者同优先级的静态广播接收器，按照注册顺序。
  • 静态：先扫描的大于后扫描的，动态：先注册的大于后注册的。

也可以动态设置优先级别
